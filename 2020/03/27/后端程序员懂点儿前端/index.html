one. js、jquery和ajax的三角恋

1、JS是一门前端语言。

2、Ajax是一门技术，它提供了异步更新的机制，使==客户端与服务器间交换数据==而非整个页面文档，实现页面的==局部更新==。

3、jQuery是一个框架，它==对JS进行了封装，使其更方便使用==。jQuery使得JS与Ajax的使用更方便

关系比喻：

若把js比作木头，那么jquery就是木板（半成品）

jQuery与ajax都是js的一个框架，各有各的功能，若js是父亲的话，jquery与ajax就是两个儿子

two. 使用jQuery

jQuery在线手册，可供随时查看：http://jquery.cuishifeng.cn/

  • jQuery核心函数：$/jQuery

作为一般函数调用： $(param)

  1. 参数为函数：当DOM加载完成后，执行此回调函数
  2. 参数为选择器字符串：查找所有匹配的标签，并将它们封装成jQuery对象
  3. 参数为DOM对象：将DOM对象封装成jQuery对象
  4. 参数为HTML标签字符串（用得少）：创建标签对象并封装成jQuery对象

作为对象使用： $.xxx()

  1. $.each()：隐式遍历数组
  2. $.trim()：去除两端的空格
  • jQuery核心对象：执行$()返回的对象.

==注意：$(‘#btn’) 一定要加“#”，很容易忽略并且使人郁闷的一个bug==

jQuery对象内部包含的是DOM元素对象的伪数组（可能只有一个元素）

jQuery对象拥有很多有用的属性和方法，可以更方便地操作DOM

  1. 基本行为
    1. size()/length ：伪数组的元素个数
    2. [index]/get(index) ：根据索引获取元素（index从0开始）
    3. each() ：遍历元素，each()函数内部有一个参数（回调函数），可以对遍历的元素进行操作
    4. index() ：返回元素的索引值

伪数组：其实是Object对象，但是有Length属性，数值下标属性

three. 创建一简单的jQuery项目

$(function () {
//$(‘#btn2’)返回一个对象
var $btn2 = $(‘#btn2’); //jQuery对象，有$符号标识
$btn2.click(function () { //给btn2绑定点击监听
var username = $(‘#username’).val();
alert(username);
})
})

$：jQuery作为函数，括号里面是作为参数的一个回调函数。

four. 使用jQuery核心函数

  • 选择器：‘#id’（原来这就是选择器啊～）

$(selector)作用：

根据选择器规则，在整个文档中查找所有匹配的标签组成了一个伪数组（可能有n个），并封装成jQuery对象

  1. 基本选择器

    1. ‘#id’
    2. element
    3. .class
    4. *
    5. selector1, selector2, selectorN (并集)
    6. Selector1 selector2 selectorN (交集)

  2. 层次选择器：查找子元素，后代元素，兄弟元素的选择器

    1. Ancestor descendant （空格➡️后代）
    2. Parent > child （>大于号➡️孩子）
    3. Prev + next （+ next表示后面的，兄弟）
    4. Prev ~ siblings（siblings：兄弟姐妹，匹配prev元素后面所有的兄弟姐妹）
